## 业务知识

### 指标定义

| 指标 | 全称 | 定义 | 计算公式 |
|------|------|------|----------|
| GMV | Gross Merchandise Value | 商品交易总额 | Σ(订单金额) |
| AOV | Average Order Value | 平均订单金额 | GMV / 订单数 |
| ROAS | Return on Ad Spend | 广告回报率 | 收入 / 广告花费 |
| 转化率 | Conversion Rate | 访客转化为买家的比例 | 订单数 / 访客数 × 100% |

### 维度说明

| 维度 | 说明 | 示例值 |
|------|------|--------|
| 品类 | 商品分类 | 电子产品、服装、家居... |
| 渠道 | 销售渠道 | Amazon、独立站、eBay... |
| 地区 | 销售区域 | 北美、欧洲、亚太... |
